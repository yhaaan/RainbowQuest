# RainbowQuest
Unity6 내부의 에셋만 활용하여 12시간의 제한시간 동안 제작한 1인 개발 게임입니다.  

---
### 조작법
* Q , E      : 색 전환
* Left Mouse : 공격
* W,A,S,D    : 이동
